# 贪吃蛇游戏

这是一个使用 HTML5 Canvas 和 JavaScript 实现的经典贪吃蛇游戏。

## 功能特点

- 使用 WASD 键控制蛇的移动
- 400x600 像素的游戏区域
- 实时显示得分
- 历史最高分排行榜（前三名）
- 游戏结束时显示最终得分
- 支持重新开始游戏

## 如何运行

1. 克隆仓库：
```bash
git clone https://github.com/JonasJoe233/Demo-VibeCoding101-Part1.git
```

2. 进入项目目录：
```bash
cd Demo-VibeCoding101-Part1
```

3. 使用 Python 启动本地服务器：
```bash
python3 server.py
```

4. 在浏览器中访问：
```
http://localhost:3000
```

## 游戏控制

- W 键：向上移动
- S 键：向下移动
- A 键：向左移动
- D 键：向右移动

## 技术栈

- HTML5
- CSS3
- JavaScript
- Canvas API
